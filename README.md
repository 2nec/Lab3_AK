KEY LIST
  - -h or --help             
  - -v or --version                   
  - -n or --name
  - -l[<val1>,...] or --list=[<num1>,...]

Program options: 

  -h or --help                          - messages with program options

  -v or --version                       - current version of file

  -n or --name                          - current file name

  -l[<val1>,...] or --list=[<num1>,...] - list of available arguments
